The Code for Hundo Wonder

                        
                            //get values from the page
                            //controller function where it starts
                            function getValues() {
                                //get values from the page
                                let startValue = document.getElementById('startValue').value;
                                let endValue = document.getElementById('endValue').value;
                               
                            
                            
                                //we need to validate input
                                //parse into integers
                                startValue = parseInt(startValue);
                                endValue = parseInt(endValue);
                            
                                if(Number.isInteger(startValue) && Number.isInteger(endValue)){
                                    //we call generateNumbers
                                    let numbers = generateNumbers(startValue, endValue);
                                    //we call displayNumbers
                                     displayNumbers(numbers);
                            
                                }else{
                                    Swal.fire(
                                        {
                                            icon: 'error',
                                            title: 'Oops!',
                                            text: 'Please enter ONLY integers!'
                                        }
                                    );
                                }
                            }
                            
                            //generate numbers from startValue to the endValue
                            //logic function(s)
                            function generateNumbers(sValue, eValue) {
                                let numbers = [];
                            
                                //we want to get all numbers from start to end -- LOOP
                            
                                for (let index = sValue; index <= eValue; index++) {
                                   //this will execute in a loop until index = eValue
                                   numbers.push(index);   
                                }
                               return numbers;
                            }
                            
                            //display the numbers and mark even ones bold
                            //display or view function
                            function displayNumbers(numbers) {
                            
                                let templateRows = "";
                                for (let index = 0; index < numbers.length; index++) {
                                    
                                    let className = "";
                                    let number = numbers[index];
                            
                                    if(number % 2 == 0){
                                        className = "even";
                                    }else{
                                        className = "odd";
                                    }
                                    templateRows += `${number}`;
                                    
                                }
                                document.getElementById('results').innerHTML = templateRows;
                            }
The code is structured in three functions
function getValues()

is the controller function. First, the program gets the values from the page via getElementById. Next, it validates that those values are useable as integers. If they are not integers, a sweet alert will pop up for the user to let them know they need to input integers for this program to work.

function generateNumbers()

is the logic function. First, the program declares numbers to be an empty array. Next, a for loop executes until the index of the array is equal to the endValue that the user input. Then it returns the Array of Numbers.

function displayNumbers()

is the view function. First, the program declares the templateRows as an empty string so that nothing is populated in it at the beginning. Next, a for-loop runs again until the user input's endValue -- however, in this display function, classNames are included to differentiate the styling of 'even' and 'odd' rows. In order to figure out if a row is 'even' or 'odd', if / else statements are used with a modulo operator (%) which calculates the remainder during division. So -- if there is a remainder of 2 it means it is divisible by even numbers. If there is not a remainder of 2 - it is divisible by odd numbers. Using this, in addition to classNames, the program can run different styling for even/odd rows based on the classes assocuated with them. In this code, this is highlighted with the template literal and embedded expression using the dollar sign and curly braces. The string and placeholders are passed into the function. Lastly, the results ID is called via innerHTML and templateRows so that it displays on the page.
